=== How to use ===

== For Windows

Download DirectShow for Java wrapper and place dsj.{dll|jar} in
.\lib\phybots\library\ directory.
http://www.humatic.de/htools/dsj.htm

Then, launch picode.cmd as follows:

.\picode.cmd -type Human
.\picode.cmd -type MindstormsNXT -address btspp://deadbeaf

== For Mac OS X

./picode.sh  -type Human
./picode.cmd -type MindstormsNXT -address btspp://deadbeaf

=== Supported hardware ===

- Mindstorms NXT (Windows, Mac)
- Kinect for Windows (Windows)
